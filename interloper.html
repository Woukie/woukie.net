The Internet Interloper project

The Internet Interloper project lets you leave notes on any page of the internet, visible to anyone else using the extension.

To add a note, right click anywhere on a page and select 'Add Note'.

Update username and auth code

If you steal/guess someone else's auth code and put it here, you can post as them. So please don't give away your auth code or make it easy to guess.

How it works

  • When visiting a page, the extension asks the backend for a list of notes for the URL. If there are notes, the extension displays them on the page.
  • When adding a note, the extension sends the note to the backend, which saves it in a database. Each note has a position, text, url, and a userID, obtained from an auth code sent with the request.
  • The user data stored on the backend is an ID, auth code and username.
